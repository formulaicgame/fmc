# Fmc 
![picture](.github/picture.png)

# How to run
```
cd client && cargo run --release
```
# Controls
| keys             | description                                                                 |
| ---------        | -----------                                                                 |
| `escape`           | Pause                                                                       |
| `WASD`             | Move                                                                        |
| `spacebar`         | Jump/Fly, double tap to toggle flight                                       |
| `shift`            | Fly down                                                                    |
| `control`          | Speed up horizontal flight                                                  |
| `e`                | inventory                                                                   |
